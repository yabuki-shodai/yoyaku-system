# 06. 未決事項・今後の検討課題

- 管理者権限に段階（オーナー/一般管理者など）を設けるか。
- 予約の変更・キャンセル可能期限（例: 前日まで、当日不可 等）をどう設定するか。
- 予約無断キャンセル（ノーショー）に対するペナルティ運用の要否。
- メール通知の送信基盤（SMTP/外部サービス）の選定。
- 予約変更・キャンセルの監査ログ（誰が・いつ変更したか）を残す必要性。
- 同時に対応可能な予約数の上限（席数相当）の運用ルール。
- 複数店舗展開・オンライン決済・LINE通知連携など将来拡張の優先順位。
- 認証方式の詳細（自前JWT実装か、Auth0等の外部IDaaS利用か）。
