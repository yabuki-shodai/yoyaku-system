# 04. 画面一覧（案）

## 4.1 一般ユーザー向け（apps/user）

| 画面 | 概要 |
|---|---|
| トップページ | 店舗紹介・メニュー抜粋・予約導線 |
| 会員登録 | メールアドレス・パスワード・氏名・電話番号入力 |
| ログイン / パスワード再設定 | 認証関連 |
| メニュー一覧 | 施術メニュー・料金・所要時間の一覧 |
| 予約作成（メニュー選択） | 1つ以上のメニューを選択 |
| 予約作成（日時・スタッフ選択） | カレンダーから空き枠を選択、担当スタッフ指定（任意） |
| 予約確認・確定 | 選択内容の確認と確定ボタン |
| 予約完了 | 完了メッセージ、予約内容の表示 |
| マイページ（予約一覧） | 今後の予約・過去の予約履歴 |
| 予約詳細 / 変更 / キャンセル | 個別予約の詳細確認・変更・キャンセル操作 |
| プロフィール編集 | 氏名・電話番号・メール等の編集 |

## 4.2 管理者向け（apps/admin）

| 画面 | 概要 |
|---|---|
| ログイン | 管理者認証 |
| ダッシュボード | 当日/週間の予約状況サマリ |
| 予約一覧（カレンダー/リスト） | 日別・スタッフ別の予約状況、検索・絞り込み |
| 予約詳細 / 新規登録（代理） / 編集 / キャンセル | 予約の詳細確認・電話予約の代理登録・変更 |
| 顧客一覧 | 顧客の検索・一覧 |
| 顧客詳細 | プロフィール・予約履歴・メモ |
| メニュー一覧 / 登録 / 編集 | メニューのマスタ管理 |
| スタッフ一覧 / 登録 / 編集 | スタッフのマスタ管理、稼働時間設定 |
| 営業設定 | 営業時間・休業日・予約枠単位の設定 |

## 4.3 補足
- 画面遷移・ワイヤーフレームの詳細は本ドキュメントのスコープ外とし、別途デザインフェーズで検討する。
